Tag "ul", "ol" e "li"

A tag "ul" representa lista não ordenada

Ex:

  • Arroz
  • Feijão
  • Farinha
  • Margarina
  • Óleo

Já a tag "ol" representa uma lista ordenada

Ex:

  1. Arroz
  2. Feijão
  3. Farinha
  4. Margarina
  5. Óleo

A tag "li" representa o item da lista como mostrado no exemplo acima.

Tag "font"

A tag "font" é utilizada para modificar a fonte do texto (cor, estilo de letra, tamanho, decoração).

Ex:

Vamos adicionar o estilo de letra Sans-Serif a esta fraze: Estou estudando HTML.

Vamos adicionar cor a esta fraze: Estou estudando HTML.

Vamos adicionar um tamanho de letra para esta fraze, exemplo tamanho 6: Estou estudando HTML.

Aparentemente essas técnicas não são mais utilizadas diretamente pelo HTML, para isso utilizamos o CSS com a tag "style".
